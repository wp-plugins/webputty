Integrate WebPutty.net with WordPress

== Description ==

The WebPutty Helper for WordPress plugin makes it easy to integrate [WebPutty](http://www.webputty.net) with WordPress.

For more information, please see the [plugin home page](http://soderlind.no/archives/2011/12/20/wordpress-and-webputty/)

== Installation ==

= Requirement =
* PHP: 5.2.x or newer

= Manual Installation =
* Upload the files to wp-content/plugins/webputty/
* Activate the plugin

= Automatic Installation =
* On your WordPress blog, open the Dashboard
* Go to Plugins->Install New
* Search for "webputty helper"
* Click on install to install the plugin
